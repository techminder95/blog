# Nuxt Supabase Auth

> Basic authentication example using Nuxtjs and Supabase

## Functionality

- Register
- Login
- Logout
- Automatically redirect unauthenticated users

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```
